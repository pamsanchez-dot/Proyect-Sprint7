# Análisis de vehículos en USA

## Descripción
Esta aplicación web permite explorar datos de anuncios de venta de coches en Estados Unidos mediante visualizaciones interactivas.

## Funcionalidad
- Histograma del kilometraje de vehículos
- Gráfico de dispersión entre precio y kilometraje

## Cómo usar
1. Abre la aplicación en el siguiente enlace:
https://tu-app.onrender.com

2. Selecciona las casillas para visualizar los gráficos.

## Tecnologías utilizadas
- Python
- Pandas
- Streamlit
- Plotly Express
